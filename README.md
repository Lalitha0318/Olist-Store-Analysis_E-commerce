# 🛒Olist-Store-Analysis_E-commerce

**Power BI • Tableau • SQL • Excel • MySQL**

This project presents an end-to-end analysis of Olist e-commerce data, leveraging Excel, SQL, Power BI, and Tableau. The dashboard processes 99K+ orders and $16M revenue to provide insights into sales trends, customer segmentation, and key performance metrics, enabling data-driven business strategies.

## **Objectives**

➤ Analyze sales and revenue trends

➤ Understand customer distribution and behavior

➤ Evaluate product category performance

➤ Track delivery performance and efficiency

➤ Monitor key e-commerce KPIs

## **Tools & Technologies**

➤ **Excel** – Data Cleaning & Preparation

➤ **SQL** – Data Analysis & Querying

➤ **Power BI / Tableau** – Dashboard & Visualization

## **Key KPIs**

## **Dataset Overview**

The dataset used in this project is sourced from the Olist E-commerce platform and contains detailed information about orders, customers, products, payments, reviews, and logistics. It is a relational dataset with multiple interconnected tables, enabling comprehensive analysis of the e-commerce supply chain.

## **Core Tables Description**

➤ **olist_orders_dataset**
Contains order-level information such as order ID, purchase timestamp, delivery dates, and order status

➤ **olist_customers_dataset**
Includes customer details like customer ID, location (city, state), and unique identifiers

➤ **olist_order_items_dataset**
Provides product-level details for each order, including product ID, seller ID, price, and freight value

➤ **olist_products_dataset**
Contains product information such as category and product attributes

➤ **olist_sellers_dataset**
Includes seller details such as seller ID and location

## **Supporting Tables**

➤ **olist_order_payments_dataset**
Captures payment details including payment type, installments, and payment value

➤ **olist_order_reviews_dataset**
Contains customer review scores and feedback for each order

➤ **olist_geolocation_dataset**
Provides geographic data (zip code, city, state) for mapping and regional analysis

➤ **product_category_name_translation**
Maps product category names from Portuguese to English for better understanding

## **Data Characteristics**

➤ Covers 99K+ orders across multiple regions

➤ Includes customer, product, seller, and payment data

➤ Enables end-to-end e-commerce analysis

## **Data Relationships**

The dataset follows a relational model, where:

Orders connect customers, payments, reviews, and order items

Order items link products and sellers

Geolocation data enables regional insights

## **Key Insights**

➤ Weekday sales ($11M) are significantly higher than weekend sales ($3M)

➤ Top-performing category: Beleza Saúde generating highest revenue

➤ São Paulo (SP) contributes the highest sales among all states

➤ Average delivery time is ~12 days, impacting customer satisfaction

➤ Lower shipping days generally correlate with higher review scores
